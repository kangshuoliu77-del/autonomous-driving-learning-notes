# 🎮 能控性与能观性

---

## 🔹 能控性

**核心问题** 输入 $u$ 能否把状态 $x$ 从任意起点送到任意终点

**简化判断** 能从任意起点送到原点 $\Leftrightarrow$ 能从任意起点送到任意终点

**能控性矩阵**

$M = [B \quad AB \quad A^2B \quad \cdots \quad A^{n-1}B]$ 横着拼

**判据** $\text{rank}(M) = n \Leftrightarrow$ 系统能控

满秩 $\to$ 能控 降秩 $\to$ 不能控

---

## 🔹 能控性例子

**能控** $A = [0 \ 1; 0 \ 0]$ $B = [0; 1]$

$AB = [1; 0]$ $M = [0 \ 1; 1 \ 0]$ $\det(M) = -1 \neq 0$

**不能控** $A = [1 \ 0; 0 \ 1]$ $B = [0; 1]$

$AB = [0; 1]$ $M = [0 \ 0; 1 \ 1]$ $\det(M) = 0$

第一个状态 $\dot{x}_1 = x_1$ 与 $u$ 无关 永远控制不了

---

## 🔹 能观性

**核心问题** 输出 $y$ 能否反推出状态 $x$

**能观性矩阵**

$N = [C; CA; CA^2; \cdots; CA^{n-1}]$ 竖着堆

**判据** $\text{rank}(N) = n \Leftrightarrow$ 系统能观

---

## 🔹 能观性例子

**能观** $A = [0 \ 1; -2 \ -3]$ $C = [1 \ 0]$

$CA = [0 \ 1]$ $N = [1 \ 0; 0 \ 1]$ $\det(N) = 1 \neq 0$

**不能观** $A = [-1 \ 0; 0 \ -2]$ $C = [1 \ 0]$

$CA = [-1 \ 0]$ $N = [1 \ 0; -1 \ 0]$ 第二列全 $0$ 秩 $=1 < 2$

---

## 🔹 对比总结

| | 能控性 | 能观性 |
|:---|:---|:---|
| 问题 | $u \to x$ 能控制吗 | $y \to x$ 能看到吗 |
| 矩阵 | $M = [B \ AB \ \cdots]$ 横拼 | $N = [C; CA; \cdots]$ 竖堆 |
| 判据 | $\text{rank}(M) = n$ | $\text{rank}(N) = n$ |

---

## 🔹 对偶原理

系统 $(A,B,C)$ 能观 $\Leftrightarrow$ 对偶系统 $(A^T, C^T, B^T)$ 能控

能控性和能观性本质是同一个问题

---

## 🔹 小结

- 能控性 输入能否控制状态 看 $M$ 的秩
- 能观性 输出能否反映状态 看 $N$ 的秩
- 两者通过对偶原理联系
