# 📐 状态空间表达式

---

## 🔹 核心公式

**状态方程** $\dot{x} = A x + B u$  描述系统动态

**输出方程** $y = C x + D u$  描述观测输出

**状态空间表达式** 状态方程 + 输出方程

---

## 🔹 系统矩阵维度

| 符号 | 维度 | 名称 |
|:---|:---|:---|
| $x$ | $n \times 1$ | 状态矢量 |
| $u$ | $r \times 1$ | 输入矢量 |
| $y$ | $m \times 1$ | 输出矢量 |
| $A$ | $n \times n$ | 系统矩阵 |
| $B$ | $n \times r$ | 输入矩阵 |
| $C$ | $m \times n$ | 输出矩阵 |
| $D$ | $m \times r$ | 直接传递矩阵 |

---

## 🔹 拉普拉斯变换

**核心性质**

$\mathcal{L}[\dot{y}(t)] = s Y(s) - y(0)$

$\mathcal{L}[\ddot{y}(t)] = s^2 Y(s) - s y(0) - \dot{y}(0)$

**作用** 把微分方程变为代数方程 求导变成乘以 $s$

**传递函数** 初始条件为零时 $G(s) = Y(s) / U(s)$

---

## 🔹 极点与稳定性

**极点** 传递函数分母 $=0$ 的根

**为什么重要** 时域解是 $e^{pt}$ 的组合 $p$ 是极点

| 极点实部 | 稳定性 |
|:---|:---|
| 全部 $< 0$ | 渐近稳定 $e^{-t}$ 衰减到 $0$ |
| 有 $> 0$ | 不稳定 $e^t$ 增长到 $\infty$ |
| 有 $= 0$ 单根 | 临界 常数 |
| 有 $= 0$ 重根 | 不稳定 线性增长 |

---

## 🔹 零点

**零点** 传递函数分子 $=0$ 的根

不决定稳定性 但影响瞬态响应形状 无零点平缓上升 有零点可能超调

---

## 🔹 实现问题

已知传递函数 写出状态空间表达式 $\dot{x}=Ax+Bu$ $y=Cx+Du$

**情况一 无零点 $m < n$**

微分方程 $y^{(n)} + a\_{n-1}y^{(n-1)} + \cdots + a\_1\dot{y} + a\_0 y = b\_0 u$

取状态变量 $x\_1 = y$ $x\_2 = \dot{y}$ $\dots$ $x\_n = y^{(n-1)}$

- $\dot{x}\_1 = x\_2$
- $\dot{x}\_2 = x\_3$
- $\cdots$
- $\dot{x}\_{n-1} = x\_n$
- $\dot{x}\_n = -a\_0 x\_1 - a\_1 x\_2 - \cdots - a\_{n-1} x\_n + u$

$y = b\_0 x\_1$

$A$ 为友矩阵 最后一行是 $-a\_0, -a\_1, \dots, -a\_{n-1}$  
$B = [0 \ 0 \ \cdots \ 0 \ 1]^T$  
$C = [b\_0 \ 0 \ \cdots \ 0]$ $D = 0$

**情况二 有零点 $m \le n$**

传递函数 $W(s) = (b\_n s^n + \cdots + b\_0)/(s^n + a\_{n-1}s^{n-1} + \cdots + a\_0)$

方法一 $A,B$ 同上 $C = [\hat{b}\_0 \ \hat{b}\_1 \ \cdots \ \hat{b}\_{n-1}]$ 其中 $\hat{b}\_i = b\_i - a\_i b\_n$ $D = b\_n$

方法二 $A$ 同上 $C = [1 \ 0 \ \cdots \ 0]$ $D = b\_n$ $B = [\beta\_{n-1}; \cdots; \beta\_0]$ $\beta$ 按递推公式计算

---

## 🔹 小结

- 状态空间表达式 = 状态方程 + 输出方程
- 拉普拉斯变换把微分方程变代数方程
- 极点实部全负则稳定
- 无零点选输出为状态 $A$ 为友矩阵 有零点两种实现方法
